Readlist




Descrição


Readlist foi o projeto final do curso CS50x,
foi utilizado nesse projeto HTML, CSS, Python e SQL.
Nesse site um usuário pode criar uma conta e botar os livros que ele leu,
o site vai pegar toda a informação e gerar estatísticas sobre os livros,
autores mais lidos ou livros mais antigos por exemplo.
